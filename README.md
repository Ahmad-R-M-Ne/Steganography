## Network Steganography Messenger

> **Hide. Encrypt. Transmit. Communicate.**

🌐 **Network Steganography Messenger** is an security-research project that explores covert communication techniques over standard network protocols.

📦 Messages are encrypted using **ChaCha20-Poly1305**, fragmented into secure chunks, and transported through carefully selected network packet fields within an authorized laboratory environment.

⚡ Built for protocol engineering, packet analysis, cryptography research, and blue-team validation.

### 🚀 Key Features

* 🔒 End-to-end authenticated encryption
* 🌐 ICMP-based transport architecture
* 📡 Packet-level covert communication research
* 🧩 Message fragmentation and reassembly engine
* 📊 Reliability layer with ACK/NACK mechanisms
* 🛡️ Defensive analysis and detection validation
* 📝 Interactive CLI messenger interface
* 🔬 Designed for education, research, and security laboratories

### 🎯 Project Mission

To demonstrate how modern cryptography, network protocols, and steganographic techniques can be combined into a measurable, observable, and defensively analyzable communication platform.
